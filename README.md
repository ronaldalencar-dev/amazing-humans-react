# Amazing Humans

Amazing Humans is a modern storytelling platform where readers and writers connect. It is a completely free platform that allows authors to publish their work and readers to discover new, original fiction.

## Features

- **Standard & Interactive Stories:** Authors can publish traditional chapter-by-chapter books or create immersive, branching interactive stories where readers choose their own path.
- **Bulk PDF Upload:** Writers can effortlessly upload entire manuscripts at once using PDF files.
- **Community Driven:** Readers can follow authors, leave reviews, rate chapters, and engage in the comment section.
- **Real-time Notifications:** Users are notified instantly when a followed author publishes a new chapter.
- **Secure Authentication:** Robust user authentication with a custom 6-digit email verification flow.

## Tech Stack

- **Frontend:** React, Vite, TailwindCSS, React Router
- **Backend & Database:** Firebase (Firestore, Authentication, Cloud Functions)
- **Deployment:** Firebase Hosting
